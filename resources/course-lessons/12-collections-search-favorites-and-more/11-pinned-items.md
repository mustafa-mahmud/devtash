# Pinned Items

We have a field in the items table called `isPinned`. I want to make it so we can pin then with the icon and then those will get put to the beginning of the listings page for that item as well as on the dashboard.

## Feature Spec

Copy the course file `context/features/pinned-spec.md` to your project and run:

```text
/feature load pinned-spec.md
```

Now run `/feature start`.

Check pinning functionality then run:

```text
/feature test
/feature review
/feature complete
clear
```

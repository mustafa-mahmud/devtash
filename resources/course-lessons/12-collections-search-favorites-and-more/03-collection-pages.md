# Displaying Collection Items

So right now we can add items to collections, but there is no way to see which collections they use. I want to do a few things in this lesson:

1. Create the /collections page to show all collections
2. Create the /collections/[id] page to show the items in the collection

Let's run the following prompt:

```text
/feature load create the /collections page and show the collections

Create the /collections/[id] page to show the items in that collection

Use the existing cards

Link the "View all collections" in the sidebar to /collections and link all collection cards to that specific collection page
```

Run `/feature start`

Check the pages and iterate if needed

After iterating and testing, run:

```text
/feature review
/feature complete
/clear
```

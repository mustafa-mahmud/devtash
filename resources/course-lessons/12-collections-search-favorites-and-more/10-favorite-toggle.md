
# Favorite Toggle

We have the /favorites page. Now we need to be able to toggle items and collection favorites. This is easy enough to just add a manual prompt for the feature.

Run the following prompt:

```text
/feature load Add a favorite button to the drawer, collection page and cards to toggle it as a favorite.
```

Then run `/feature start`

Check to make sure that you can add and remove favorites.

```text
/feature test
/feature review
/feature complete
clear
```

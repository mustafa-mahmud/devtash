# Use Playwright To Test CRUD

We tested CRUD manually, but let's have the Playwright MCP. You could install Playwright with npm and then set up full E2E testing, but I don't think that's really necessary here. Plus I want to give you an example of using the Playwright MCP. 

Run the following prompt: 

```text
Use the Playwright MCP to test CRUD functionality. Log in with the user "demo" at devstash.io with the password "12345678". Click on the "New Item" button in the top bar and add a new snippet. Then proceed to edit the title of that snippet and update it. Finally, delete the snippet. 
```

It should open up the browser and do as we asked.

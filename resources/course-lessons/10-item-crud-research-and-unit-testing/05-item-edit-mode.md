# Item Edit Mode

We have our drawer. Now I want to be able to edit an item. I have a spec file for you in the course files. Move `item-drawer-edit-spec.md` to your project's `/context/features` folder.

Let's run the following prompt:

```text
/feature load item-drawer-edit-spec.md

```

Run through the workflow:

```text
/feature start
/feature test
/feature review
/feature complete
```

It should create an action and also add a test for it.

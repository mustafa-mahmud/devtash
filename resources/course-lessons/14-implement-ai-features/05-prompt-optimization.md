# Prompt Optimization Feature

I want to have a feature to help users optimize their AI prompt items. Let's use a simple load prompt for this one:


```text
/feature load Implement prompt optimization for the prompt item types. It should look at the current prompt and refine if needed, then ask if the user wants to use the updated prompt. Put the "Optimize" button in the header, much like the "Explain" button on snippets and commands
```

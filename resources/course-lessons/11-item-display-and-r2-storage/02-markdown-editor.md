# Markdown Editor

For our snippets and commands, we used a code editor. For notes and prompts, we want to use a markdown editor.

There is a spec file for this. Move `markdown-editor-spec.md` to your project's features folder and run the following command:

```text
/feature load markdown-editor-spec.md
```

Now run it with `/feature start`.


Iterate if needed. Make sure that the preview works and is easily accessed.

Once finished, run:

```text
/feature test
/feature review
/feature complete
/clear
```

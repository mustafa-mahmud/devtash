# Image Display

I want the images to display a bit differently than the rest of the types.

I want images to be an image grid with a lightbox.

Create the file `context/features/image-display-spec.md` and add the course file content.

Let's run the following:

```text
/feature load image-display-spec.md

/feature start
```

Test in the browser and make sure it is to your liking as far as the image sizes, padding, modal size, etc. Test the delete as well.

Once you are finished, run the following:

```text
/feature test
/feature review
/feature complete
/clear
```

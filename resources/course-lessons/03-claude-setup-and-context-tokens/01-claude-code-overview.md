## What is Claude Code?

Claude Code is Anthropic's command-line tool for agentic coding using the Claude LLM. It runs directly in your terminal and can read your files, write code, run commands, and basically work alongside you as an AI pair programmer.

This is different from Claude.ai, which is a chatbot that runs in the browser. Claude Code has direct access to your file system. You can use something like Claude.ai or ChatGPT for helping your write code, but Claude Code, Cursor, Codex, They're not just answering questions for you to copy and paste. It's actually doing the work and can run tools on your machine.

Claude Code is also available as a VS Code extension. I'll be using this later after we get set up.

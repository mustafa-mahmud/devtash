# VS Code Extension

You have a few options when using Claude Code. You can run it in the terminal like we have been doing, but there is also a VS Code extension. You can use this extension even if you are using Cursor or Windsurf, because those are just forks of VS Code.

## Installation

To install, just like any other extension, click on the extension tab on the sidebar and search for "Claude Code". Then install it.

From there, you can hit `ctrl+shift+p` and search for "Open New Claude Code Tab". Each tab will have its own context.

## Differences

There are some differences.

When you use the VS Code extension, you are limited to the project folder, which in most cases, including this one is fine.

There are also some commands that don't work with the extension at least right now. I think the one that is most important that is missing is `/rewind`, which can bring you to an earlier state in your chat. The really important commands like `/context` and `/clear` work fine.

The pros of the extension are that it has a much better looking display of everything. It's more clear what Claude is doing. The diff view is great. You can clearly see what code is being added and what is being removed. This is the reason I like to use it.

### Context Percentage

Also in the command bar, you have a context usage monitor, which is very helpful. It will tell you how much you have used and have left.

### Think Button

You also have a button to use think mode if you are stuck on something.

So I'll be using the extension for the remainder of the course, but that doesn't mean that you have to.
